Important FAQ

What is the Document Object Model(DOM)?

Document Object Model (DOM) is a cross-platform and language-independent interface that treats an HTML or XML document as a tree structure wherein each node is an object representing a part of the document. The DOM represents a document with a logical tree.

How do you select an element from DOM?

We can select an element by taking it's id or by class name or by tag name or or by css selectors or get all matching elements.

What is the event delegation in the context of DOM and why is it useful?

Event Delegation is a pattern based upon the concept of Event Bubbling. It is an event-handling pattern that allows you to handle events at a higher level in the DOM tree other than the level where the event was first received.It also Reduces the number of event listeners,simplifies handling events for dynamically added elements and centraizes event handling logic to make code easier.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes and styles using the DOM, we can:
1.Change and remove existing elements in the DOM.
2.Create and add new elements to the page.
3.Change the styles for elements.
4.Add event listeners to the elements to make them interactive.
5.To change the style of an HTML element, use the syntax: `document.getElementById(id).style.property = new style`.
6.Use `innerHTML` to set the HTML content inside an element, `style` to access CSS styles, and `textContent` to set the text content of an element.
